famodulus - Demo Application

Welcome to the famodulus demo application!

On this site, you can try out the functions offered by the famodulus-client library and compare performance with calculations performed directly in the browser.

By default, calculations performed in the browser use the leemon BigInt library. famodulus-client can however detect if the Verificatum JS crypto library has been loaded by the browser and will use it if available.

To try out the functions offered by famodulus-client, go to the following pages:

  • Direct for direct, unblinded outsourcing of one or multiple modexp(s)
  • DEC for outsourcing modexps using basic decomposition algorithms

For further information, the API documentation for famodulus-client is also provided.
