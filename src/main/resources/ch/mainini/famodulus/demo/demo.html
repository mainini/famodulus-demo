Outsourcing Demo

On this page, outsourcing of a single or multiple modexps can be tried out and performance can be compared to calculations performed in the browser. After choosing the outsourcing method, enter the parameters for the modexp(s) separated by comma. When calculating multiple modexps, default parameters can be set which apply in case they are missing for a modexp.

Direct outsourcing requires a single server and simply sends base, exponent and modulus as is to the server without any blinding.

Outsourcing with basic decomposition (DEC) algebraically blinds either base or exponent or both, requiring two or four servers depending on the blinding.
